Este é o trabalho 3 de POO, onde será implementado um sistema de biblioteca.

Instruções para a execução do código:

1. Rodar a classe "Main" ou apenas executar o .jar executável localizado em "Trab3_POO_jar".
2. Executar as funções do trabalho como desejar.

Sobre a execução:
Na inicialização, aparecera uma interface gráfica onde existirão 7 botões clicáveis. Cada um desses botões realiza uma função descrita no trabalho.
Após ter executado algumas dessas funções, deve-se clicar no botão "Encerrar Programa", para que as informações dos usuarios e livros possam ser salvas no arquivo.

Notas :
As informações da biblioteca são salvas num arquivo "information.csv", que será criado no mesmo diretorio onde o programa for executado. Esse arquivo deve existir inicialmente pois ele sempre é lido ao se inicar o programa.
As datas do sistema são salvas na forma da quatidade de segundos que se passaram desde o ano de 1970 ( pra facilitar as operações com as datas).
Liste os livros e os usuarios caso queira saber o ID de cada um para um emprestimo.
